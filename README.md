# 수학 모험 게임 🎮

초등학교 3학년 학생들을 위한 곱셈과 나눗셈 학습 게임입니다.

## 🎯 프로젝트 소개

이 게임은 초등학교 3학년 학생들이 곱셈과 나눗셈을 재미있게 배울 수 있도록 설계된 교육용 게임입니다. 
게임화(Gamification) 요소를 통해 학습 동기를 부여하고, 시각적 도움 기능으로 수학 개념을 쉽게 이해할 수 있습니다.

## ✨ 주요 기능

### 🎮 게임 모드
- **연습 모드**: 곱셈/나눗셈을 자유롭게 선택하여 연습
- **난이도 선택**: 쉬움/보통/어려움 3단계
- **문제 수 조절**: 5개~20개까지 선택 가능

### 📚 학습 기능
- **시각적 도움**: 블록, 과일, 동물 등으로 문제를 시각화
- **힌트 시스템**: 단계별 힌트 제공
- **실생활 문제**: 일상생활과 연관된 문제 출제
- **오답 노트**: 틀린 문제를 저장하고 다시 풀기

### 🎁 보상 시스템
- **코인 획득**: 정답을 맞추면 코인 획득
- **경험치 시스템**: 레벨업을 통한 성장
- **아이템 상점**: 코인으로 캐릭터 꾸미기 아이템 구매
- **업적 시스템**: 다양한 업적 달성

### 📊 학습 관리
- **학습 대시보드**: 전체 학습 통계 확인
- **연산별 통계**: 곱셈/나눗셈 각각의 정답률 추적
- **난이도별 분석**: 난이도별 성취도 확인
- **연속 학습일**: 매일 학습 습관 형성

## 🛠️ 기술 스택

- **Frontend**: React 18+ with TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **State Management**: Zustand
- **Routing**: React Router v6
- **Build Tool**: Vite

## 📦 설치 및 실행

## 📖 사용 방법

1. **시작하기**: 이름을 입력하고 게임을 시작합니다.
2. **연습 모드 선택**: 홈 화면에서 "연습 모드"를 선택합니다.
3. **설정 선택**: 연산 타입(곱셈/나눗셈), 난이도, 문제 수를 선택합니다.
4. **문제 풀기**: 숫자 키패드를 사용하여 답을 입력합니다.
5. **결과 확인**: 게임 종료 후 결과와 보상을 확인합니다.

## 🎓 교육 성취기준

### 세 자리 수 범위의 곱셈
- **[4수01-04]** 곱하는 수가 한 자리 수 또는 두 자리 수인 곱셈의 계산 원리를 이해하고 그 계산을 할 수 있다.

### 세 자리 수 범위의 나눗셈
- **[4수01-05]** 나눗셈이 이루어지는 실생활 상황과 연결하여 나눗셈의 의미를 알고, 곱셈과 나눗셈의 관계를 이해한다.
- **[4수01-06]** 나누는 수가 한 자리 수인 나눗셈의 계산 원리를 이해하고 그 계산을 할 수 있으며, 나눗셈에서 몫과 나머지의 의미를 안다.

## 🎨 주요 화면

### 1. 환영 화면
- 사용자 이름 입력
- 게임 시작

### 2. 홈 화면
- 캐릭터 정보 (레벨, 경험치)
- 보유 코인 및 통계
- 메뉴 선택 (연습 모드, 대시보드, 상점 등)

### 3. 게임 플레이 화면
- 문제 표시
- 시각적 도움
- 숫자 키패드
- 힌트 버튼
- 진행 상황 표시

### 4. 결과 화면
- 별점 (1~3개)
- 정답률, 평균 시간
- 획득한 코인 및 경험치
- 문제별 상세 결과

### 5. 학습 대시보드
- 전체 통계
- 곱셈/나눗셈 별도 통계
- 난이도별 정답률
- 학습 팁

### 6. 아이템 상점
- 캐릭터 꾸미기 아이템
- 카테고리별 분류 (모자, 의상, 악세서리, 배경)
- 구매 및 착용 기능

### 7. 업적
- 달성한 업적 확인
- 진행 중인 업적 진행도
- 업적 보상

### 8. 오답 노트
- 틀린 문제 목록
- 정답 및 힌트 표시
- 다시 풀기 기능
